#  3D printing widgets

Simple JavaScripts to calculate things for your 3D printer.

https://tomcatmwi.github.io/threaded_insert_calculator/index.html

##  Threaded insert calculator

Calculates optimal and passthrough hole diameters for threaded inserts

##  Bolt passthrough calculator

Calculates the diameter of holes for various bolts to pass through unimpended

# How to use?

Both calculators return the exact value to use in your CAD application, so your 3D printed hole will correspond to what you actually need.

# How can you help?

For each material, a range of shrinkage figures was established. If you disagree with these values, or would like to add a new material, please let me know or make a PR.
